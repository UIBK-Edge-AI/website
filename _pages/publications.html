Setup Required for Real API Access

This page attempts to fetch real publication data from the UIBK database. Due to browser CORS restrictions, you have several options:

Option 1: Use a CORS Proxy (Recommended for Testing)

Install and run a local CORS proxy:

npm install -g cors-anywhere
cors-anywhere

This will run a proxy server at http://localhost:8080

Option 2: Server-Side Integration (Recommended for Production)

Create a backend endpoint that fetches from UIBK and serves the data to your frontend.

Option 3: Browser Extension

Use a browser extension like "CORS Unblock" for development (not recommended for production).
